# 图片相似度搜索

这是一个简单的图片相似度搜索工具，可以从图库中找出与输入图片最相似的图片。

## 功能特点

- 使用ResNet50预训练模型提取图片特征
- 使用余弦相似度计算图片间的相似程度
- 支持常见图片格式（PNG、JPG、JPEG、GIF、BMP）

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 将你的图片库放在 `images` 文件夹中
2. 运行程序：
   ```bash
   python image_search.py
   ```
3. 根据提示输入要搜索的图片路径
4. 程序会显示最相似的图片及其相似度分数

## 注意事项

- 确保图片库中的图片格式正确
- 第一次运行时会下载预训练模型，需要保持网络连接
- 图片库较大时，首次加载可能需要一些时间
